What is this?

This small web app fetches list of all of your owned games, along with some metadata, and exports it into a spreadsheet for easier filtering, categorization, and sharing. Inspired by Kris Ligman's 'game tasting' project and steam's incredibly bare-bones library filtering tools.

Currently exports: steam store link, name, total playtime, linux playtime, mac playtime, and windows playtime (note that platform-specific timers have been added by valve relatively recently, and thus may be incomplete or may be missing completely for some titles).

I plan to eventually expand the functionality to also fetch the game details from the store api (publisher, developer, release date, tags, genres, etc.).

How does this work?

Written in python, using flask, flask-openid, requests, and pyexcel.

Authentication is handled through OpenID - valve handles your credentials directly on their site, and will redirect you afterwards to the app with your account's steamID encoded in the url. This id is then used to perform a lookup of owned games through steam's web api (api.steampowered.com/IPlayerService/GetOwnedGames/).

Source code available at github.com/rmmbear

Is my account information retained on the server?

No. Your steamid is kept client-side, in a session cookie, which is cleared after looking up owned games. That list is kept on the server up until the spreadsheet has been generated.
